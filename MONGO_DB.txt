MONGO_DB:

-> Tables are referred as collections and schema as document in Mongo DB and here users is collection

DB -> To create/use DB -> 
	use schoolDB 
	db.createCollection("students") -> this creates the collection in DB
	db.students.insertOne({ name: "Alice", grade: "A" })
	show collections -> this lists all the collections
CREATE -> db.users.createOne({}) - single user, db.users.createMany([{}]) - multiple users
READ -> db.users.find({}) - to view all documents, db.users.find({key:value}) - to view particular documents
DELETE -> db.users.deleteOne({key:value}) - single user, db.users.deleteMany([{}]) - multiple users,
	  db.users.updateOne({unique key:valye},{$unset:{key}}) -> to remove particular fields, can add multiple 	  keys in same line to remove multiple fields	
	  
	**db.reservations.remove({ $and : [ {"city" : "London"}, {"starts_at" : "2020-06-18T20:13:58"} ] })
		
UPDATE -> db.users.updateOne({unique key:value},{$set:{key:value}}) - multiple users


To connect MONGO with DB:

const dbURL = "URL" //url where your db server is hosted
const dbName = "name" //name of the db which you wanted to connect and use
const connection = dbURL + dbName
mongoose.connect(connection).then()  //Mongoose is a library which connects Node JS and Mongo DB
